※ 누락 시 무한 로딩 에러가 발생합니다.
2 제목 및 이미지 설정
3 페이지 활성화 설정

체크 해제 시 해당 메뉴와 페이지가 HTML에서 완전히 제거됩니다.

배경 안의 캐릭터들을 마우스로 드래그하여 배치하거나, 우측 하단의 흰색 점을 당겨 크기를 조절하세요.

📖 구글 스프레드시트 100% 활용 가이드

이 가이드는 백엔드(서버) 역할을 하는 Apps Script와 시트의 사용법을 상세히 설명합니다.
아래 단계에 따라 세팅하면 데이터가 자동으로 웹 대시보드에 연동됩니다.

📋
준비: 스프레드시트 템플릿 복사

가장 먼저, 모든 자동화 스크립트가 포함된 원본 시트를 내 구글 드라이브로 복사해야 합니다.

1. 아래 링크를 클릭하여 템플릿 시트에 접속합니다.

[CR] 월간 보고서 자동화 템플릿 열기 🔗

전체 URL: https://docs.google.com/spreadsheets/d/1KXPR7_BfTx7uj0RtJZ2r-UhCVVrKUt_W17Efq0rehVI/edit?usp=sharing

2. 내 구글 드라이브에 저장

좌측 상단의 [파일] → [사본 만들기]를 클릭하여 내 드라이브에 저장한 후, 복사된 시트에서 아래 1단계부터 진행하세요.

📌
1단계: API 서버(Web App) 배포하기

웹페이지가 시트의 데이터를 읽어갈 수 있도록 통로를 열어주는 필수 과정입니다.

1. Apps Script 창 열기

구글 스프레드시트 상단 메뉴에서 [확장 프로그램] → [Apps Script]를 클릭합니다.

2. 웹 앱(Web App)으로 배포

우측 상단의 파란색 [배포] → [새 배포]를 클릭합니다.
유형을 '웹 앱(Web App)'으로 선택하고, 액세스 권한(Who has access)을 반드시 '모든 사용자(Anyone)'로 변경한 뒤 [배포]를 누릅니다.

3. 권한 허용 및 URL 복사

보안 경고창이 뜨면 `고급 → 안전하지 않음으로 이동`하여 권한을 허용합니다.
완료 후 나타나는 웹 앱 URL (https://script.google.com/macros/s/.../exec)을 복사하여, 생성기(빌더)의 1번 항목에 붙여넣으세요.

🤖
2단계: 유튜브 및 기본 보고서 자동화

시트 상단의 [🤖 보고서 자동화] 메뉴를 통해 동작합니다.

최근 데이터 수집 (1개월 / 6개월)

클릭 시 지정된 공식 유튜브 채널의 영상 목록, 조회수, 좋아요, 댓글을 유튜브 API를 통해 즉시 긁어와 유튜브 영상 상세 데이터 시트에 쌓습니다.

지표 시트 새로고침

API 연동 없이, 시트에 이미 쌓인 데이터를 바탕으로 월별 통계를 재계산하고 최신순으로 정렬합니다.

매월 마지막 날 자동 수집 켜기

이 기능을 켜두면 매달 말일 자정에 스크립트가 자동으로 실행되어, 이번 달의 유튜브 성과 데이터를 알아서 백업해 둡니다.

👥
3단계: 크리에이터 관리 자동화

시트 상단의 [👥 크리에이터 관리] 메뉴를 통해 동작합니다.

1. 채널 ID/이름 자동 완성 편의기능

크리에이터 설정 탭의 C열에 크리에이터의 유튜브 링크(또는 @핸들명)만 적어두고 이 버튼을 누르세요. 스크립트가 URL을 추적하여 정확한 채널 ID(A열)와 채널명(B열)을 자동으로 채워줍니다.

2. 최근 1개월 / 3개월 영상 수집

등록된 모든 크리에이터의 채널을 스캔하여, 지정한 기간 내에 올라온 모든 영상을 긁어옵니다.
결과는 크리에이터 데이터 탭에 최신 날짜가 맨 아래로 오도록 예쁘게 정리됩니다.

🌍
4단계: 외부 SNS (인스타그램 & 틱톡) 연동

시트 상단의 [🌍 SNS 연동 (인스타/틱톡)] 메뉴를 통해 동작합니다.

API 정보 설정 (최초 1회 필수)

각 플랫폼(Meta Developer, TikTok Developer)에서 발급받은 Access Token을 입력하는 메뉴입니다. 입력된 토큰은 스크립트 속성에 안전하게 비공개로 저장됩니다.

데이터 수집 실행 (1개월 / 3개월)

저장된 토큰을 이용해 내 계정에 업로드된 릴스, 게시물, 틱톡 영상의 시청수/좋아요/댓글 수를 불러와 각각의 시트에 자동 기록합니다.

📱
5단계: 센서타워 데이터 정리

센서타워 데이터 정리 실행

센서타워(원본) 탭에 센서타워에서 다운로드한 Raw 데이터를 날짜 구분 없이 쭉 붙여넣기만 하세요.
상단의 [📱 센서타워 정리] 메뉴를 누르면, 스크립트가 날짜별로 DAU 등을 합산하고 예쁘게 가공하여 센서타워(최종) 탭으로 자동 이동시켜 줍니다.
